# Region meta data

Dataset with meta information on countries and their regional groupings.

## Usage

``` r
regions_all
```

## Format

A tibble, one row per country. Variables include

- division_numeric_code:

  Divison numeric code

- name_country:

  Name country

- cluster:

  Name of cluster

- subcluster:

  Name of subcluster

- is_unmarried_sexual_activity:

  SA0 indicator
