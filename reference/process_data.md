# Process survey data

Process survey data

## Usage

``` r
process_data(dat, regions_dat, start_year = 1970, is_married = TRUE)
```

## Arguments

- dat:

  Survey data

- regions_dat:

  File with region meta data

- start_year:

  Filter out data prior to this year

- is_married:

  Logical, select married women data?

## Value

Processed survey data
