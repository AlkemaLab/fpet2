# Plot Epsilon Innovations

Plot Epsilon Innovations

## Usage

``` r
plot_eps(fit, is_married, estimates_df, add_trad = FALSE)
```

## Arguments

- fit:

  fit object from a global fit

- is_married:

  boolean

- estimates_df:

  dataframe of estimates

- add_trad:

  boolean, whether to add traditional epsilon innovations

## Value

NULL, saves plots to fit\$output_dir/eps.pdf
