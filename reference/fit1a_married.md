## Format

\## \`fit1a_married\` list with global fit objects including

- post_summ:

  data frame with posterior summaries for relevant parameters

- geo_unit_index:

  data frame with geo_unit info

- source_index:

  data frame with sources of survey data with indices used in the model

- time_index:

  data frame with geo_unit info

- other settings:

  spline settings, t_star, etc
