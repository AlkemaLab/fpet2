# Get national data for subnational fitting

Get national data for subnational fitting

## Usage

``` r
get_nat_for_fitting(national_dat_df, survey_df_use)
```

## Arguments

- national_dat_df:

  National survey data

- survey_df_use:

  Subnational survey data used in fitting

## Value

Dataframe with national data not in subnational survey data
