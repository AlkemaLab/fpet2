# Ad hoc calculation of CP Traditional

Taken from UNPD/fpemlocal: try to calculate cp traditional from any and
modern

## Usage

``` r
ad_hoc_calculate_cp_trad(obs)
```

## Arguments

- obs:

  data frame with contraceptive_use_any, contraceptive_use_modern,
  contraceptive_use_traditional

## Value

data frame with contraceptive_use_traditional filled in where possible
